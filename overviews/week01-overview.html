This week we will study

  • The structure of a web page
  • HTML tags and elements
  • Lists
  • Resources

You will need your book the first week. You should be able to purchase it at the BC Bookstore, an online bookstore, or use Books 24x7.

We will also discuss setting up an individual web hosting account so your assignments and projects can be displayed via the Internet. This should be done by Week 2, since assignments after that are required to be uploaded to the web. Web hosting accounts are usually available for free.

Objectives

  1. Learn the difference between tags, attributes and elements
  2. Create a web page using an editor of your choice. I use WebStorm.
  3. Demonstrate an understanding of basic text tags such as <P> and <H1>
  4. Describe the value of semantic mark-up

Readings

Become familiar with the MDN pages on Web development.

  • Web Development on MDN
  • HTML
  • CSS

I'm not going to stand over you and say you must read this or that. Instead, I ask that you spent between 30 and 60 minutes just browsing these pages on MDN. There are many interesting tutorials, and many pages that serve as references. You can always find the links to specific information on Google, but if you know how to navigate this crucial site, it will pay you back handsomely.

For instance, read HTML basics this week, and CSS first steps next week:
  • HTML Basics
  • CSS First Steps
