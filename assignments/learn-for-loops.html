Overview

In this lesson we will learn about using for-loops in a browser. We have been writing a lot of code at the command line with node. But now we want to also focus on using the same code in a browser.

Create Directory

  • Inside it create a file called index.html and index.js.
  • Link in a CSS file of your choice
  • Initialize index.html with the emmet html5 or ! macro.
    • Set the title to be Learn about for-loops.
    • Create a header element and put an h1 element inside it. Set the content of the h1 element to Learn for-loops. Add a by-line.
    • Create an empty main element
    • Create a footer with a copyright notice in it and a link back to your home page..

for Loop Basics

We had a good chance to see how for-loops work in the Looping over Complex Types assignment. In this assignment, we will expand on that knowledge by once again visiting the subject of responding to button clicks by performing specific actions. In particular, we will dynamically add items to a list each time a button is clicked.

Append to List

Here is the code for appending text to a list:

const elfCode = {
    appendToList: (list, value) => {
        const li = document.createElement("li");
        li.appendChild(document.createTextNode(value));
        list.appendChild(li);
    }
};
