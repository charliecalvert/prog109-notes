Looping over Complex Types

The simple types are number, string, boolean, null, and undefined. These types are essential to JavaScript programming and you must understand them.

There are two important complex types that you must know. The good news is that they are not really that complex, they are not really that hard to understand. The types are:

  • Arrays
  • Objects

Both these types are commonly used in loops, so we will cover that subject at the same time.

Arrays

Here is how to declare an array:

    const myArray = [];

We can also pre-populate our array with content:

    const numbers = [1, 2, 3, 4, 5, 8, 10];

Here is how to access a memeber of this array using zero based indexers

    const numbers = [1, 2, 3, 4, 5, 8, 10];
    console.log(numbers[1]);
    console.log(numbers[0]);
    console.log(numbers[numbers.length - 1]);

Here is how to dynamically add an item to an array:

    const numbers = [1, 2, 3, 4, 5, 8, 10];
    numbers.push(12);
    console.log(numbers);
    numbers.pop(); // Remove the last item
    console.log(numbers);

As you will see later, you can put numbers, strings, booleans, arrays, and objects in inside an array. For instance:

    const items = [1, 'two', [3, 4], false, null];

Loop over Arrays

Here is how to loop over an array:

    const items = [1, 'two', [3, 4], false, null];
    for (let i = 0; i < items.length; i++) {
        console.log(items[i]);
    }

Here is a simpler way to loop over an array:

    const numbers = [1, 2, 3, 4, 5, 8, 10];
    for (let number of numbers) {
        console.log(numbers);
    }

Here are JavaScript methods for converting a string to an array and vice-versa:

function arrayToString(array) {
    console.log(array.join(', '));
}

function stringToArray(input) {
    console.log(input.split(','));
}


arrayToString(["bundle.js", "*.js.map", ".c9"]);

stringToArray("bundle.js, *.js.map, .c9");

Objects

Objects contain properties and functions. For now, we will only add properties to our objects.

Here is how to declare an object:

    const myObject = {};

Here is how to declare an object with two properties:

    const myObject = {
        five: 5,
        six: 'six'
    };

Here is how to access a member of this object

    const myObject = {
        five: 5,
        six: 'six'
    };            

    console.log(myObject.five);
    console.log(myObject.six);

Here is how to dynamically add an item to an object:

    const myObject = {
        five: 5,
        six: 'six'
    };            

    myObject.newItem = 'This is a new item in my object.'
    console.log(myObject.five);
    console.log(myObject.six);
    console.log(myObject.newItem);
