More on Emmet

Type in these Emmet commands and press tab to expand them:

  • p
  • h2
  • ul
  • ul>li
  • ul>li*5

Press CTRL+SPACE to see Emmet options if they do not pop up automatically.

Select SHIFT-CTRL-P to bring up a list of all commands. Try searching for Format Document and then press ENTER. The abbreviation for the format command is CTRL-SHIFT-I.

TIP: Use CTRL-SHIFT-E to switch between the editor and the explorer.

The Favicon

Here is how to draw the favicon:

<link rel="icon" type="image/png" href="favicon.png">

Create an icon

  • Open Windows Paint
  • Select Home | Resize
  • Uncheck Aspect Ratio and select Pixels
  • Set the width and height to 32
  • Be creative
  • Save your work as favicon.png and put it in the root of your repository.

TIP: On the view page zoom in and select gridlines
