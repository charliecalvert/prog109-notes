More on Emmet

Type in these Emmet commands and press tab to expand them:

  • p
  • h2
  • ul
  • ul>li
  • ul>li*5

Press CTRL+SPACE to see Emmet options if they do not pop up automatically.

Select SHIFT-CTRL-P to bring up a list of all commands. Try searching for Format Document and then press ENTER. The abbreviation for the format command is CTRL-SHIFT-I.

TIP: Use CTRL-SHIFT-E to switch between the editor and the explorer.
