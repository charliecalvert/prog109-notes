Create an icon

  • Open Windows Paint
  • Select Home | Resize
  • Uncheck Aspect Ratio and select Pixels
  • Set the width and height to 32
  • Be creative
  • Save your work as favicon.png and put it in the root of your repository.

TIP: On the view page zoom in and select gridlines
