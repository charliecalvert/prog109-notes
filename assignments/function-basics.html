Show Results in Html

In function-basics.html create an H1 element with the content Function Basics.

Add three empty paragraph (P) elements. Give them the following IDs:

  • simpleFunctionDisplay
  • functionParametersDisplay
  • functionReturnDisplay

Add a button with the content Call Functions and the ID set to functionCallsAction

<button id="THE ID GOES HERE">THE CONTENT GOES HERE<button>

In function-basics.js, inside an window.onload method, respond to clicks on the button by creating a function that calls all three functions that you created earlier. It should display the output in the appropriate paragraph element.

Here are some hints. First, be sure all your code is inside the window.onload method:

                
window.onload = () => {
    // All code goes in here.
}

First find the elements on the page that we are going to need to access. This is often the first step in this kind of code. We do this by calling document.getElementById. In our case, we are using this method to find the HTML paragraph elements on our HTML page so that we can use JavaScript to modify their content. 


const simpleFunctionDisplay = document.getElementById('simpleFunctionDisplay');
const functionCallsAction = document.getElementById('functionCallsAction');

Declare or reuse your simple function, accessing the textContent property of the paragraph. This will set the content of the P element. In other words, your goal is to display text in one of the empty paragraph elements on the HTML page. For example, your code might look a little, but not exactly, like this: 


function simple() {
    console.log('simple');
    simpleFunctionDisplay.textContent = "simple";
}

NOTE: We have a slight preference for textContent over innerHTML as it is faster and a bit safer. But if you want to insert HTML into an element rather than plain text, then use innerHTML.

Finally, tell the button to call our function when it is clicked


functionCallsAction.onclick = simple;

Note that we will need to modify the other functions so that, in addition to the console.log statements, they also set the textContent of each paragraph in function-basics.html.

Note also that we don't call a function when we assign it to onclick. In other words, we write this: onclick = simple not onclick = simple(). The latter has side effects that you probably don't want in that it executes the simple function at the time of the assignment, rather than when the button is clicked.

Let's make the program a bit more interesting by allowing the user to enter text, and to insert the text into our list when the button is clicked.

Add a label and an input control to our HTML file:

<label for="nameDisplay">Enter a name</label>
<input type="text" id="nameDisplay" name="nameDisplay">

Notice how we connect the label to the input control via the for attribute of the label and the id attribute of the input element.

We get data from the input like this:

const data = nameDisplay.value;

Add to List Background

Sometimes we have files that we use in multiple places in our code. One such a file is elf-code.js from the ListButton assignment. As you may know, here is the contents of that file:

const elfCode = {
    appendToList: (list, value) => {
        const li = document.createElement("li");
        li.appendChild(document.createTextNode(value));
        list.appendChild(li);
    }
}

We also need to remember how button onclick events work:

const someAction = document.getElementById('someAction');
someAction.onclick = function() {
    // Handle the onclick event in here.
}

Add to List Steps

Link your JavaScript files into function-basics.html.

<script src="./elf-code.js"></script>

In function-basics.html also add an unordered list

<ul id="listDisplay"></ul>

To make this work we need several things to happen:

  • We need handles to three HTML elements. Get the handle by calling getElementById. Get handles to:
    • The button: addToListAction
    • The list element: listDisplay
    • The input element: nameDisplay
  • Inside the onclick method of the addToListAction button:
    • Use the value property of the nameDisplay to get the input entered by the user
    • Use elfCode.appendToList to append the user input to the listDisplay

For more information on input controls and the value property, see this stackoverflow discussion. I'm expecting you to use method 1 of those outlined in the discussion.
