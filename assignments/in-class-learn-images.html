The goal of this assignment is to learn about working with images. We will create a page with at least two images in it that are hosted on Wikimedia. The images should not be larger than 800px in width

Create a file called in-class-learn-images.html in the root of your repository.

Here is a sample IMG element:

<img class="picture" src="image.jpg" alt="something nice to look at">

Read more about IMG on MDN

The Favicon

Here is how to draw the favicon:

<link rel="icon" type="image/png" href="favicon.png">

Create an icon

  • Open Windows Paint
  • Select Home | Resize
  • Uncheck Aspect Ratio and select Pixels
  • Set the width and height to 32
  • Be creative
  • Save your work as favicon.png and put it in the root of your repository.

TIP: On the view page zoom in and select gridlines
