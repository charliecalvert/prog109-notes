It's time to create our first web page. It will be a simple html page about a movie you enjoy.

Base everything on HTML5 standards and make sure you text looks fairly okay on both a desktop and a mobile device such as your phone. (We will refine our presentation later, for now, just produce something reasonable.)

Here is the HTML5 boilerplate. Don't copy the whole thing, but if you are looking for examples of how to complete a particular tag, this is a good resource. Let me say it again, just to be clear. Don't copy the whole index.html page from HTML5 Boilerplate. Instead, just look at the particular tags referenced in this assignment. For instance, when I'm asking for the html tag with the lang set to English, HTML5 Boilerplate has it almost right:

<html class="no-js" lang="">

But let's skip the no-js and use English as the language:

<html lang="en">

The charset and viewport tags are exactly right. Again, I just want to see the tags I explicitly call out below. Don't include other tags.

Make it simple, so long as it meets the requirements discussed in class. There will be an example provided during class.

At least meet the following guidelines:

  • Use an opening html tag the lang set to English. More on MDN.
  • Add a meta charset UTF8 tag.
    • On StackOverflow. A bit more from StackOverflow.
  • Use a body and head tag
  • Add a title tag
  • Write the name of your favorite movie using the largest h tag
  • Write the name of the director of the movie and make the name italic with em tag.
  • Write a very brief description of the moving in a paragraph tag
  • Bold at least one word with the strong tag.
  • Do not use a break tag
  • Use an hr tag
